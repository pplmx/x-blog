X-Blog Architecture

FastAPI + Next.js 16 Blog System • SQLite + React 19 + Tailwind CSS

Backend API (12 Routers)

  • • posts: CRUD, slug, publish, views
  • • comments: create, reply, IP tracking
  • • categories: nested structure
  • • tags: post-tag many-to-many
  • • search: full-text post search
  • • stats: views + likes counter
  • • admin: JWT-protected operations
  • • rss, sitemap, health, export, upload

Frontend (Next.js 16)

  • • App Router with TypeScript 6
  • • React 19 + Tailwind CSS
  • • 40+ reusable components
  • • Mermaid, Markdown, Dark mode
  • • MSW mock for API testing
  • • Vitest 113 test cases
  • • Biome CI linting
  • • SEO: robots.txt, sitemap, RSS

Data Model

  • • Post: title, slug, markdown, excerpt
  • • Category: 1:N → Posts
  • • Tag: N:N ↔ Posts (post_tags)
  • • Comment: 1:N thread (parent_id)
  • • SQLite (SQLAlchemy ORM)
  • • Pydantic v2 schemas
  • • Rate limiting: SlowAPI
  • • Cache: TTL-based in-memory
