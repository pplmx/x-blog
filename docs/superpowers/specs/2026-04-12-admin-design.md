# X-Blog 后台管理设计

**日期**: 2026-04-12  
**状态**: 已批准

## 1. 技术栈

- **FastAPI Admin** - 后台管理框架
- **JWT 认证** - 内置用户认证
- **SQLAlchemy** - 沿用现有模型

## 2. 功能

| 功能     | 描述                 |
| -------- | -------------------- |
| 登录认证 | JWT 方式的用户登录   |
| 文章管理 | 创建、编辑、删除文章 |
| 分类管理 | 创建、编辑、删除分类 |
| 标签管理 | 创建、编辑、删除标签 |

## 3. 路由

| 路径                | 功能         |
| ------------------- | ------------ |
| `/admin`            | 管理后台首页 |
| `/admin/auth`       | 登录页面     |
| `/admin/posts`      | 文章 CRUD    |
| `/admin/categories` | 分类 CRUD    |
| `/admin/tags`       | 标签 CRUD    |

## 4. 实现

- 安装 `fastapi-admin` 依赖
- 创建用户模型 (User)
- 配置 Admin 实例
- 注册模型到 Admin 面板

## 5. MVP 范围

- 仅后台管理，无前台用户系统
- 使用 SQLite 存储管理员账号
